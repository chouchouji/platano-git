# platano-git

> A tool library for simplifying **git** operations

## user's guidance

### Install

#### Using [npm](https://www.npmjs.com/):

```
npm install platano-git -g
```

#### Using [yarn](https://yarnpkg.com/):

```
yarn global add platano-git
```
